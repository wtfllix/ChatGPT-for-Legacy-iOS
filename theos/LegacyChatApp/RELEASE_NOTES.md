# Legacy-Chatbox 0.1.0

First Theos-based release for jailbroken legacy iOS devices.

## Highlights

- Supports iPhone 4 / 4S, iPhone 5, and iPad 4 on iOS 6 in portrait.
- Adds OpenAI-compatible provider configuration with multiple saved profiles.
- Adds text chat, local conversation history, streaming responses, and temporary thinking/reasoning display.
- Adds optional image input for multimodal-capable providers.
- Adds editable system prompt in Settings.
- Adds lightweight Markdown readability support.
- Adds iOS 6-style UI, launch images, app icon, and About page.

## Known Limits

- Markdown rendering is lightweight and not a full Markdown engine.
- Image input requires provider-side support.
- Streaming behavior depends on provider SSE compatibility and old-device performance.
- Data is stored locally on device; there is no cloud sync.
